# Architecture Thinking Flow Cheat Sheet

## Core Reasoning Process

Architecture thinking follows this flow:

1. Domain Concern (business goal)
2. Architecture Characteristics (-ilities)
3. Architecture Decisions

---

## Example 1

Domain Concern:
Black Friday traffic

Architecture Characteristics:
- Scalability
- Performance
- Availability

Architecture Decisions:
- Load balancers
- Caching
- Distributed infrastructure

---

## Example 2

Domain Concern:
Fast feature releases

Architecture Characteristics:
- Deployability
- Testability
- Maintainability

Architecture Decisions:
- CI/CD pipelines
- Automated testing
- Modular architecture

---

## Example 3

Domain Concern:
Financial transactions

Architecture Characteristics:
- Security
- Auditability
- Reliability

Architecture Decisions:
- Encryption
- Secure APIs
- Logging and monitoring

---
